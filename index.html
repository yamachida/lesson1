HTML & CSS

ページの作成に使用言語です。HTMLとCSSを組み合わせることで、Webなページを身につけることができます。

jQuery

素敵な動きを手軽に実装できるJavaScriptライブラリです。 アニメ効果をつけたり、Ajax（エイジャックス）を使って外部ファイルを長く使ったりといろいろなことができます。

ルビー

オープンソースの動的なプログラミング言語で、シンプルさと高い生産性を備えています。大きなWebアプリケーション小さな日用ツール、さまざまなソフトウェアを作ることができます。

PHP

HTMLだけではページの内容を添えることはできません。PHPはHTMLにプログラムを詰め込み、それを可能にします。
